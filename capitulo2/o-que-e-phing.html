O que é Phing

Voltar

Phing é uma ferramenta de automatização baseada no Apache Ant (Apache Ant). Você pode fazer qualquer coisa com Phing que você faria com uma ferramenta tradicional, como o GNU make (gnumake), o Phing utiliza XML para construir arquivos e classes para a execução de tarefas.

Pelo Phing ser baseado no Apache Ant, partes deste manual também são adaptados a partir do manual do Ant. Somos extremamente gratos ao projeto Ant por criar (e continuar mantendo) uma ferramenta de automatização inspiradora, e também ao seu licenciamento de código aberto que torna possível que nós aprendamos uns com os outros e que possamos construir cada vez ferramentas melhores.
