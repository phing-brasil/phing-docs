XML e Phing

Um arquivo válido para usar com o Phing possui a seguinte estrutura :

  • O prólogo do documento

  • Exatamente um elemento chamado

  • Muitos elementos do Phing (< property>, < fileset>, etc)

  • Um ou mais elementos possuindo tarefas do Phing ou tarefas definidas pelo usuário (< install>, < bcc> etc)
